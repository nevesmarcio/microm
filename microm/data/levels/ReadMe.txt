Level Editor Supported features:

## Recognized IDs ##
[board]
Determina a zona limitrofe do nível

[goal]
É neste ponto que o level é terminado com sucesso

[spawn]
É nesta coordenada que a daBox inicia o nivel

[daBox]
Serve para determinar o tamanho do player

[ground_n]
Este tipo de objectos é inócuo para a daBox

[portal_entry_n]
Teleporta os objectos dinâmicos para o portal_exit correspondente
[portal_exit_n]
Para onde os objectos são teleportados do portal_entry correspondente

[wall_n]
Este tipo de objectos parte o daBox: são os obstáculos reais

[star]
Nesta coordenada é feito o spawn de um collectible

[message_entry_n]
Coordenada de entrada da mensagem
[message_exit_n]
Coordenada de saida da mensagem

[message_simple_n]
